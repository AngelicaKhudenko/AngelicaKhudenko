DataContainer - класс который может сохранять в себе почти любое количество данных обобщённого типа.
MainExample - реализация примера использования DataContainer.

package comparators:
1) DoubleComparator - реализация интерфейса Comparator для чисел Double
2) IntegerComparator - реализация интерфейса Comparator для чисел Integer
3) ObjectComparatoe - реализация интерфейса Comparator для объектов
4) StringAlphabeticComparator - реализация интерфейса Comparator для строк по алфавиту
5) StringLengthComparator - реализация интерфейса Comparator для строк по длине
